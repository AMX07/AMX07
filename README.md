# Hey, I'm Ansh Mittal

I'm an open-source AI researcher. I like building things from scratch to understand how they work — from search engines to autograd engines to cipher crackers.

## Recent Projects

| [autoresearch](https://github.com/AMX07/autoresearch) | AI agents conducting research on single-GPU nanochat training automatically |

| [micrograd](https://github.com/AMX07/micrograd) | Re-implementation of Karpathy's minimal autograd engine and neural net library |
| [Cipher-Cracking](https://github.com/AMX07/Cipher-Cracking) | Decrypts classical ciphers using cryptanalysis techniques |
| [digit_recognition](https://github.com/AMX07/digit_recognition) | MNIST digit recognition with Streamlit, hosted on AWS |
